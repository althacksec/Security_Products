# Orca Security

**Kategori:** CNAPP / Cloud Security  
**Geliştirici:** Orca Security  
**Fiyatlandırma:** Talep üzerine  
**Deployment:** SaaS (agentless, SideScanning teknolojisi)  
**Lisans Modeli:** Ticari, kurumsal abonelik

---

## Nedir?

Orca Security, tescilli "SideScanning" teknolojisiyle cloud workload'larını agentless olarak tarayan ve risk değerlendirmesi yapan bir CNAPP platformudur. Wiz ile doğrudan rakip konumundadır; her ikisi de agentless mimariyi benimser. Orca'nın öne çıkan yönleri: deployment kolaylığı, müşteri desteği kalitesi ve uyum izleme yetenekleridir.

PeerSpot verilerine göre Orca Security kullanıcılarının %100'ü platformu başkalarına önereceklerini belirtmektedir.

---

## SideScanning Teknolojisi

Geleneksel agent tabanlı tarama yerine, Orca cloud sağlayıcısının API'si aracılığıyla workload'ların anlık görüntülerini (snapshot) okur. Bu sayım:

- Çalışan sistemlere hiç dokunmadan tam görünürlük
- İşlem yükü sıfır; production ortamı etkilenmiyor
- Dakikalar içinde deployment

## Temel Özellikler

| Özellik | Açıklama |
|---|---|
| **SideScanning** | Agentless, non-intrusive workload görünürlüğü. |

| **AppSec** | Uygulama güvenliği ve API risk analizi. |
| **Secret Detection** | Cloud'da açıkta kalan credential ve API key tespiti. |

## Artılar

- **Deployment kolaylığı:** SideScanning ile en hızlı kurulum süreçlerinden biri.
- **Müşteri desteği kalitesi:** Wiz'e kıyasla müşteri desteği üstünlüğü kullanıcı yorumlarında öne çıkıyor.
- **Uyum odaklılık:** Sık denetlenen kuruluşlar için güçlü compliance raporlama araçları.
- **%100 tavsiye oranı:** PeerSpot verilerine göre tüm kullanıcılar öneriyor.
- **Finansal şeffaflık:** Compliance puanı olarak güvenlik duruşunu raporlama kolaylığı.
- **IaC tarama:** Infrastructure-as-code güvenlik analizi.

---

## Eksiler

- **Raporlama sınırlılıkları:** Yönetici raporları yeterince detaylı veya özelleştirilebilir değil.
- **Pazar payı:** Wiz'in gerisinde (%6.3 vs %17.4 mindshare, 2026 verisi).
- **3. taraf entegrasyon:** Wiz'e kıyasla daha az native entegrasyon.
- **Runtime koruma:** SideScanning gerçek zamanlı runtime engelleme yapamaz; tespit sonrası müdahale gerekir.
- **Kaynak kodu tarama:** Uygulama source code analizi sınırlı.

---

## Kullanım Alanları

- Uyum gereksinimlerinin yüksek olduğu sektörler (finans, sağlık)
- Hızlı cloud görünürlüğü ihtiyacı olan kuruluşlar
- CSPM ve zafiyet yönetimi odaklı cloud güvenliği
- Wiz alternatifi arayan orta ölçekli kuruluşlar
